# URL Architecture

## Rules

-   One product = One primary URL.
-   Routing generates URLs.
-   Never store full URLs in DB.
-   Product ID is permanent.
-   Slugs are unique.
-   Keep slug history.
-   301 on slug change.
-   Never reuse old slugs.

## URL example

`/ar/steam/steam-wallet-usa/steam-wallet-gift-card-10-usd`
