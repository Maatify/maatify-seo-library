# SEO Fundamentals for Backend Developers

## Goal

Understand how Google discovers, understands and indexes pages.

## Google workflow

1.  Discovery (links, sitemap)
2.  Crawling
3.  Rendering
4.  Indexing
5.  Ranking

## Backend responsibilities

-   Stable URLs
-   Correct HTTP status codes
-   Canonical
-   Structured Data
-   Fast responses
-   Clean HTML
-   Internal linking
-   XML Sitemap

## Golden rule

Every backend decision should help Google understand the page more
accurately.
